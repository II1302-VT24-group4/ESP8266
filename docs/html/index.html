📚 Libraries Used

This project uses the following libraries:

  • ESP8266WiFi (GNU)
  • Firebase-ESP-Client (MIT)
  • ArduinoJson (MIT)
  • NTPClient
  • Tone (MIT)
  • u8g2 (BSD)

Make sure to install these libraries through your Arduino IDE.
